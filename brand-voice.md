# 🎨 UI/UX 톤앤매너 매뉴얼 (Brand Voice)

- **핵심 철학**: "심플 이즈 베스트!" 조작은 한 번의 터치로 직관적이게 설계되어야 하며, 유저는 복잡한 사용법을 학습할 필요가 없어야 한다.
- **애니메이션 원칙**: 아이템 추가/삭제 시 절대 뚝뚝 끊기게 화면이 변하면 안 됨. 항상 쫀득한(`spring` 스펙) Compose 애니메이션 트랜지션을 기반으로 유저에게 생동감 있는 피드백을 주어야 함.
- **상태 처리 (에러/빈 화면 방지)**: 데이터 로딩 중이거나 위시리스트가 완전히 텅 비어있을 때는 심심한 기본 텍스트 대신 `Lottie animation`이나 예쁜 일러스트를 배치하여 빈 상태마저도 아름답게 보장할 것.
- **문구 톤 (사용자 대상)**: 너무 딱딱한 시스템 메시지는 지양하고 "현재 가격을 열심히 불러오고 있어요 🚀" 와 같이 약간 친근하면서도 신뢰감을 주는 모던 앱의 문체를 사용할 것.
