# 🚀 InsightDeal

**국내 최초 "커뮤니티 딜 발견 + 4개 쇼핑몰 가격 비교" 통합 앱**

## 📱 앱 개요

InsightDeal은 한국 쇼핑 시장의 혁신을 목표로 하는 Android 앱입니다. 기존의 핫딜 커뮤니티와 가격 추적 앱의 경계를 허무는 **2-in-1 혁신 솔루션**입니다.

### 🎯 핵심 가치 제안

**"발견 → 추적 → 비교 → 구매"** 를 한 앱에서!

- **발견**: 뽐뿌/루리웹/클리앙 커뮤니티 딜을 실시간 수집
- **추적**: 폴센트 스타일로 관심 상품 가격 추적
- **비교**: 쿠팡/11번가/G마켓/옥션 4개 쇼핑몰 가격 한눈에 비교
- **매칭**: AI가 추적 상품과 커뮤니티 딜을 자동으로 연결

## 🌟 주요 기능

### 🏠 **홈 - 딜 발견 피드**
- **정보 우선 리스트 뷰**: 핫딜에 최적화된 UI/UX
- **선택적 그리드 뷰**: 시각적 쇼핑 경험
- **실시간 4몰 가격 비교**: 쿠팡/11번가/G마켓/옥션
- **스마트 필터**: 무료배송/해외제외/쿠팡보다싼딜 등
- **원클릭 추적**: "추적 추가" 버튼으로 즉시 위시리스트 등록

### 📋 **추적 - 개인 위시리스트**
- **URL 붙여넣기**: 폴센트 방식 상품 추가
- **목표가 설정**: 원하는 가격 도달 시 푸시 알림
- **가격 히스토리**: 7일/30일/90일 선택 가능
- **미니 그래프**: 카드에서 30일 가격 변동 즉시 확인
- **4몰 동시 추적**: 어느 쇼핑몰이 가장 싼지 실시간 비교

### 🎯 **매칭 - AI 딜 발견**
- **스마트 매칭**: 추적 중인 상품이 커뮤니티에 특가로 등장 시 알림
- **절약 계산**: "목표가보다 2만원 더 싸게 나왔어요!"
- **자동 검증**: 진짜 특가 vs 가짜 할인 구분

### ⚙️ **설정 - 개인화**
- **4가지 컬러 테마**: Orange/Blue/Green/Purple
- **AMOLED 최적화**: 완전한 블랙 테마 지원
- **알림 설정**: 목표가 달성/딜 발견/가격 변동별 세밀 조정

## 🏆 기존 앱들과의 차별화

### **vs 폴센트**
- 폴센트: 쿠팡만 추적
- **InsightDeal**: 쿠팡+11번가+G마켓+옥션 **4몰 비교**

### **vs 뽐뿌/루리웹**
- 기존: 딜만 구경
- **InsightDeal**: 딜 발견 → **즉시 추적** → 더 쌀질 때 알림

### **vs 다나와/에누리**
- 기존: 정적 가격 비교
- **InsightDeal**: **실시간 추적** + **커뮤니티 딜 연동**

## 🛠 기술 스택

### **Frontend (Android)**
- **Kotlin** - 100% Kotlin 기반
- **Jetpack Compose** - 모던 선언형 UI
- **Material Design 3** - 사용자 중심 디자인
- **Navigation Compose** - 매끄러운 화면 전환
- **Coil** - 효율적 이미지 로딩

### **Backend (FastAPI)**
- **Python FastAPI** - 고성능 비동기 API
- **PostgreSQL** - 가격 히스토리 저장
- **Docker** - 컨테이너 기반 배포
- **Selenium** - 4개 쇼핑몰 실시간 스크래핑

### **특별 기능**
- **Dynamic Theming System** - 4가지 컬러 + AMOLED 지원
- **Smart Price Matching** - 상품명 기반 자동 매칭
- **Real-time Push Notifications** - 가격 변동 즉시 알림

## ⚡ 빠른 시작

### **1️⃣ 레포지토리 클론**
```bash
git clone https://github.com/ddaeany0919/InsightDeal.git
cd InsightDeal
```

### **2️⃣ 백엔드 실행**
```bash
cd backend
docker-compose up --build
# → http://localhost:8000 에서 API 서버 동작
```

### **3️⃣ Android 앱 빌드**
```bash
# Android Studio에서
1. Open Project → InsightDeal 폴더 선택
2. Build → Rebuild Project
3. 실제 기기에서 Run
```

### **4️⃣ 기능 테스트**
```bash
# 주요 기능 확인
1. 홈 탭: 커뮤니티 딜 피드 확인
2. 추적 탭: 상품 URL 추가 테스트
3. 매칭 탭: AI 매칭 결과 확인
4. 설정 탭: 테마 변경 테스트
```
