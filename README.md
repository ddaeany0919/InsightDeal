# InsightDeal 2025

## 멀티 위시리스트 대량 관리 지원

### 주요 기능
- 여러 관심상품을 대량, 빠르고 쾌적하게 관리 (100개 이상도 OK)
- 모든 기능은 단순/직관/빠른 조작에 집중(심플이즈베스트!)

### 1️⃣ ⭐ CRITICAL (다수 아이템 필수 기능)
- 전체 가격 일괄 체크(모두 가격체크 버튼, 순차/병렬 처리, 진행률 UI)
- 정렬/필터/검색(최저가/이름/도달여부/카테고리 등 다양한 옵션)
- 멀티 선택 모드로 일괄 삭제, 일괄 가격체크(롱클릭/전체선택)
- 카테고리/그룹 분류: 탭/필터/DB에 카테고리 컬럼

### 2️⃣ 🎨 애니메이션 (다수 아이템용)
- 카드 등장/삭제/스크롤: AnimatedVisibility, animateItemPlacement, slideIn 등 적용
- 추가/삭제/Pull-to-refresh, 빈 상태 안내 등 Lottie/일러스트 활용

### 3️⃣ 📊 다수 아이템 관리 최적화
- Paging(무한 스크롤), last_checked 기준 네트워크 요청 최소화
- 대시보드 카드(전체 개수/도달수/평균/통계) 표시

### 4️⃣ 🔔 알림/백그라운드(다수 관리)
- WorkManager 예약(전체 자동 가격체크 1일 1회)
- FCM 푸시 알림(목표가 도달시)
- 아이템별 알림 ON/OFF 토글 제공

### 5️⃣ 🔐 백업/동기화(다수 아이템 보호)
- Firebase/Google Drive 연동 자동 백업/복구 지원
- 멀티 기기 데이터 실시간 동기화(앱삭제/기변/웹/태블릿/PC 모두 동시 지원)

## 적용 방법 요약

### 기본 구조
1. 앱 실행 시 모든 관심상품 LazyColumn + Paging + 애니메 + Pull-to-refresh
2. 상단 TopBar: 검색, 필터, 모두체크, 그룹별 탭/카테고리 칩
3. 카드별 체크/삭제/알림/핀고정 등 다중조작
4. ViewModel: checkAllPrices(), selectItem()/toggleSelect()/deleteSelected(), paging, filter/search 지원
5. WorkManager/FCM/Cloud 연동: 스케줄-자동알림-백업

### 주요 활용 시나리오
- 1회 클릭으로 100개 가격 모두 체크, 알림, 백업 동기화까지 한번에!
- 추가/삭제/체크 모두 부드러운 애니메이션 UX(트렌디 & 직관)
- 기기변경, 데이터복구 걱정 ZERO
