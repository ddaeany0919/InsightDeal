# InsightDeal 프로젝트 현황 및 계획 (2025-12-02)

## ✅ 완료된 작업 (Completed)

### 1. 안드로이드 앱 (Frontend)
- **커뮤니티 화면 개선 (`CommunityScreen.kt`)**:
    - 탭 네비게이션 추가 (전체, 뽐뿌, 펨코, 클리앙).
    - 커뮤니티별 핫딜 필터링 로직 구현.
    - "등록된 핫딜이 없습니다" 등 빈 상태 처리.
- **홈 화면 복구 및 개선 (`EnhancedHomeScreenApplied.kt`)**:
    - 비어있던 홈 화면 UI 구현 완료.
    - `Scaffold`, `TopAppBar`, `LazyColumn`을 사용한 대시보드 구조.
    - 새로고침 시 상단 로딩 바 (`LinearProgressIndicator`) 표시.
    - `HomeViewModel` 초기화 시 데이터 자동 로드 (`init` 블록 추가).
- **이미지 로딩**:
    - `backend/image_cache` 디렉토리의 이미지를 로드할 수 있도록 설정.

### 2. 백엔드 (Backend)
- **이미지 서빙 설정 (`main.py`)**:
    - `StaticFiles`를 사용하여 `/images` 경로로 로컬 이미지 서빙 구현.
- **커뮤니티 API 개선 (`routers/community.py`)**:
    - Mock 데이터 대신 실제 DB (`models.Deal`) 조회 로직 구현.
    - 이미지 URL에 `BASE_URL`을 자동으로 붙여서 반환하도록 수정.
    - DB 연결 실패 시 Mock 데이터로 폴백(Fallback) 처리 유지.
    - Mock 데이터에도 테스트용 실제 이미지 URL 적용.
